Párrafos

Para este ejemplo se necesita tener las etiquetas adecuadas para la accesibilidad teniendo en cuenta que se pueden poner las correctas instrucciones en la estructura.

Se deben poner, para tener el manejo secuencial del teclado la etiqueta “tabindex” con el número en secuencia que se desee.
Por otro lado, se debe de colocar el atributo “aria-lebel” con la indicación para que el usuario con discapacidad vial le sea más fácil seguir las instrucciones.



El foco se tiene que distinguir suficientemente para garantizar que las personas con visión baja puedan distinguir los componentes activos de la interfaz de usuario y los gráficos.
Además, cualquier información visual para representar el foco debe tener una relación de contarse mínima de 3:1.
